# (test) HTTP SERVER FASTAPI

## Функц. возможности:

- сохранять запись - 
- получать запись по `note_id` - 
- получать все записи из бд - 
- хранить записи между перезапусками без бд - 

## Формат

{
  "note_id": "СГЕНЕРИРОВАН АВТОМАТИЧЕСКИ СЕРВЕРОМ uuid4",
  "data": "любые данные в строковом виде"
}


## Запуск

python -m venv venv

linux - source venv/bin/activate

windows - venv/scripts/activate

pip install -r requirements.txt

uvicorn main:app --reload
-----------------------------------


## Тест( Для теста в GUI Браузера - 127.0.0.1:8000/docs)

### Создание записи(2 варианта, curl / апи в браузере)

Custom: curl -X POST http://127.0.0.1:8000/notes/ \
  -H "Content-Type: application/json" \
  -d '{"data":"hello"}'
/
API : Ручка - Save Note


### Получить все записи(2 варианта, curl / апи в браузере)


Custom: curl http://127.0.0.1:8000/notes/
/
API: Ручка - Show Notes

### Получить одну запись


Custom: curl http://127.0.0.1:8000/notes/<note_id>
/
API: Ручка - Give Note


Скрипт:
- создает пачку записей в несколько потоков - 
- считает примерное время выполнения - 
- показывает приблизительный RPS - 
- проверяет получение одной записи и всех записей -
